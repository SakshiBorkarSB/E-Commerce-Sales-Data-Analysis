# E-Commerce-Sales-Data-Analysis
This project analyzes the Brazilian Olist e-commerce dataset to uncover insights into sales trends, customer behavior, and operational performance using Python-based data analysis and visualization techniques. The objective is to analyze sales trends, understand customer behavior, identify revenue-driving product categories, and predict future monthly sales using Linear Regression.
